# Web-Based-Interactive-Neural-Nets
An interactive neural network visualizer for WebAssembly built in Rust.

# Requirements
three-d for Rust: https://github.com/asny/three-d

Any WebAssembly enabled browser (Chrome, Firefox, Etc)
